How the simulator works

  1. There are 3 rounds. Each round adds one more bag and more complexity.
  2. Review the keyword database and ad copy guide before building your plan.
  3. Choose keywords, landing page per keyword, write ad copy, and allocate budget.
  4. When trial mode is on, you can test without affecting the official round.
  5. When trial mode is off, click Submit Round. Admin will process submitted groups later.
Admin page is not shown here. Open it manually with #admin if needed.
